Документация СмИТ Биллинг 1.6

Добро пожаловать в документацию системы СмИТ Биллинг 1.6 — биллинговой платформы для операторов связи и сервис-провайдеров.

Веб-интерфейс биллинга: billing.smit34.ru
Технологический стек: Python 3.11, Django 4.2 LTS, PostgreSQL 16, Redis 7, Celery 5, FreeRADIUS 3.2.3, Docker Compose, Gunicorn, Nginx, AdminLTE 3 + Bootstrap 5.3.

Управление абонентами, тарификация, платёжные системы, личный кабинет, отчёты.

Управление сервером

Docker-контейнеры, PostgreSQL, Celery, обслуживание, удалённое управление.

Интеграция с оборудованием

NAS/BRAS, коммутаторы, VoIP-шлюзы, IPTV, абонентское оборудование.

Интеграция с СОРМ3

Передача данных по ФЗ, стандартные и пользовательские схемы СОРМ.

API

REST API v2, SOAP, Mobile API (JWT), обещанный платёж, управление абонентами. Примеры curl и JSON.
